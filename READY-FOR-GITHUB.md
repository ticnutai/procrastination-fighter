2. לחץ "New repository"

4. תיאור: `מערכת מקיפה למאבק בדחיינות - PWA ואפליקציית דסקטופ`

7. לחץ "Create repository"

### **2. חבר את הפרויקט ל-GitHub:**
```powershell

git branch -M main
```

### **3. העלה הכל:**
