🎯 **3 דרכים להפוך לתוכנה לווינדוס:**

═══════════════════════════════════════════════════════

## 🚀 דרך 1: PWA מיידית (הכי מהירה!)

פתח את `procrastination-app.html` ב-Chrome:
✅ לחץ על אייקון ההתקנה (+) בשורת הכתובות
✅ בחר "התקן"
✅ האפליקציה תופיע כתוכנה רגילה!

**יתרונות:**
- התקנה מיידית (30 שניות)
- עדכונים אוטומטיים
- עובד כמו תוכנה רגילה
- יש תפריט התחל וקיצור דרך

═══════════════════════════════════════════════════════

## ⚡ דרך 2: חבילה פשוטה (הכי קלה!)

הרץ את: `build-simple-windows.bat`

**יוצר:**
- קובץ ZIP נייד
- קיצור דרך לשולחן
- עובד על כל מחשב ווינדוס ללא התקנה

═══════════════════════════════════════════════════════

## 🔧 דרך 3: Electron מלא (הכי מקצועי!)

**אם יש לך רשאות מנהל:**
```bash
cd windows-app
npm run start  # לבדיקה
npm run dist   # לבנייה
```

**בעיית רשאות?** נסה:
```bash
# הרץ PowerShell כמנהל ואז:
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
npm run dist
```
